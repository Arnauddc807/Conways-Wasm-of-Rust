# Conway's Game of Life
### Built with 🦀🕸

made using [wasm-pack](https://github.com/rustwasm/wasm-pack) & [create-wasm-app](https://github.com/rustwasm/create-wasm-app)

## Usage
### 🛠️ Build the wasm binary
```wasm-pack build```

### 📁 Step into the www directory
```cd www/```

### 🚩 Start npm 
```npm start```
